# MonsterSurvival

## 1. 게임 컨셉

### High Concept
* 끊임없이 몰려오는 적들을 쓰러트리면서 살아남는 게임

### 핵심 메카닉
* 모든 공격은 자동으로 이루어지며 플레이어는 이동 조작에만 집중한다.
* 수많은 적들이 플레이어를 향해 다가온다.
* 적들은 점점 강해지며 플레이어도 강해지는 적들에 대항하기 위해 같이 성장한다.
* 플레이어는 공격으로 적을 쓰러트리면 그 즉시 경험치를 획득하고 레벨이 오를때마다 무기가 추가되거나 강화된다.

---

## 2. 개발 범위

### 플레이어 1종
* 사용자의 입력에 따라 그 방향으로 이동하며 레벨 업 할때마다 무기가 강화되거나 추가된다.

### 무기 3종
* 자동 타켓팅 방식의 무기 1종
* 일정 시간마다 플레이어 주위를 강하게 베는 무기 1종
* 일정 시간마다 플레이어 주위를 n초 동안 빙빙 돌며 공격하는 무기 1종

### 몬스터 3종
* 플레이어를 향해 다가오는 일반몬스터
* 이동속도가 빠르지만 체력이 낮은 몬스터
* 이동속도가 느리지만 체력이 높은 몬스터

### UI 요소
* 타이틀 화면 (게임시작, 게임 종료)
* 인게임 화면
* 팝업 화면 (무기 선택 or 무기 강화)
* 게임 오버 화면 (Restart 버튼, Back Title 버튼)

## 7. Activity 구성
* **MainActivity:** 게임의 타이틀 화면. 시작 버튼 및 간단한 UI를 호스팅
* **MonsterSurviavlActivty:** 실제 인게임 화면. BaseGameActivity를 상속받아 MainScene를 실행

---

### 8. Scene 구성 및 전환 관계
* MainScene: 플레이어, 몬스터, 총알이 활동하는 메인 씬
* LevelUpScene: 플레이어가 레벨업했을 때 화면에 나타나는 팝업창
* Scene Stack:
  1. MainScene에서 경험치가 가득 차면 scene.push()로 LevelUpScene을 스택 위에 쌓음
  2. LevelUpScene은 게임을 정지시키고 바로 직전의 previousScene 위에 불투명 한 창을 띄워서 메인 화면을 비추면서 자연스러운 일시정지를 구현
  3. 유저가 선택지를 고르면 pop()을 호출하여 씬을 제거, 즉시 MainScene으로 복귀

## 9. MainScene 핵심 Game Object

### Player
* 클래스 구성: AnimSprite 상속, IBoxCollidable을 통한 충돌 구현
* 동작/상호작용: 조이스틱의 입력값으로 상태(IDLE, WALK)를 전환하고 이동. 몬스터와 충돌 시 HP 감소 및 무적 시간 발생
* 핵심 코드: 경험치를 누적하고, 레벨업 시 pendingLevelUps를 증가시켜 Scene 관리자에게 팝업 호출을 예약하는 로직

### Weapon
* 클래스 구성: 화면에 직접 그려지지 않는 객체
* 동작/상호작용: 플레이어의 스탯을 읽어와 최종 쿨타임과 데미지를 계산.
* 핵심 코드: 쿨타임 타이머를 계산하여 World에 Projectile(투사체) 객체를 생성(world.add)하는 팩토리 역할 및 다음 레벨업 설명 텍스트를 제공

### Projectile
* 클래스 구성: Sprite 상속, IBoxCollidable, IRecyclable을 통한 충돌 및 재사용 구현
* 동작/상호작용: 생성 시 전달받은 규칙(회전, 전방 발사 등)에 따라 이동. 몬스터와 충돌 시 데미지 부여, 관통형 무기의 경우 최근에 때린 상대를 기억해두어서 1초에 60번(60fps기준) 데미지가 들어가는 현상 방지
* 핵심 코드: 수명이 다하거나 부딪히면 isDead 처리 후 자신을 recycleBin으로 반환하여 재활용하는 객체 생명주기 관리

### Monster (적)
* 클래스 구성: AnimSprite 상속, IBoxCollidable 을 통한 충돌 구현
* 동작/상호작용: 플레이어의 위치를 매 프레임 추적하여 이동. 체력이 0이 되면 플레이어에게 EXP를 제공
* 핵심 코드: 게임 플레이 시간이 지날수록 최대 체력과 이동 속도를 점진적으로 증가시켜 난이도 곡선을 형성

## 10. UX 진행 방법
1. 시작: 기본 무기 1개(마법 탄환)만 소지한 상태로 맵 중앙에서 스폰
2. 회피 및 파밍: 화면의 조이스틱을 드래그하여 다가오는 몬스터들을 회피 (무기는 자동 발사)
3. 경험치 획득: 몬스터 처치 시 화면 하단의 경험치 바가 즉각적으로 차오름을 시각적으로 확인
4. 레벨업: 바가 꽉 차면 화면이 반투명해지며 일시정지. 3개의 무작위 강화 카드 중 하나를 터치하여 캐릭터 강화
5. 반복: 팝업이 닫히면 더 강력해진 무기로 몰려오는 적들을 처치.
